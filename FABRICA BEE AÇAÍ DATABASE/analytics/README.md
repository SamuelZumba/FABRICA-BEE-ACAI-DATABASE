# Analytics

Esta pasta documenta exclusivamente a camada analítica do projeto.

Ela não repete o schema, as consultas SQL nem as regras de negócio, que permanecem nas pastas `database` e `docs`.

## Conteúdo

- `catalogo_kpis.md`: definição gerencial dos indicadores;
- `modelo_dashboard.md`: organização do painel e finalidade de cada visual;
- `data_lineage.md`: origem dos dados utilizados em cada indicador.
